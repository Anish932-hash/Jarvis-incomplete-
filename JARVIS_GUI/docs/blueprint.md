# **App Name**: Neon OS

## Core Features:

- Startup Animation: Create a startup animation with a glowing circular holographic logo made of particles that slowly coalesces into the JARVIS logo.
- Main HUD Interface: Develop a main HUD interface with neon cyan/blue elements, concentric circles, system panels, and futuristic indicators.
- Real System Readings: Display real PC system readings in a control panel interface using circular gauges, neon bars and concentric rings.
- Voice Mode Animation: Implement a pulse wave animation around the center of the HUD with circular ripples for voice mode activation.
- Real-time Data Display: Continuously fetch and display real-time CPU, GPU, RAM, disk, network, and battery data. The model has to select the appropriate tools to decide what information to surface, if the metrics exceed thresholds or become critical.

## Style Guidelines:

- Primary neon color: #00f2ff (neon cyan) for main UI elements and highlights.
- Secondary neon color: #33dcff (light blue) for accents and secondary indicators.
- Background color: #080808 (nearly black) to provide contrast for neon elements.
- Use 'Orbitron' (sans-serif) for a modern sci-fi look, suitable for headlines and labels. Note: currently only Google Fonts are supported.
- Use clean, minimalist vector icons with neon cyan outlines.
- Design a responsive 1920x1080 layout with a symmetrical sci-fi aesthetic using transparent glass UI panels.
- Implement smooth UI fade transitions, rotational motion on the centerpiece, and line tracing animations for a high-tech feel.
